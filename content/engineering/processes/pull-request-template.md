Create the `.github/pull_request_template.md` file and add the following content:

```
## Context

What is the context in which the following changes will be made?

## Story

[Jira Story Link]()

## Work done

What steps have you taken to fix the problem outlined in the context section?

## Checklist

- [ ] Tests - The code been covered by tests (70% minimum)
- [ ] Implementation - The functionality meets the acceptance criteria of the ticket
- [ ] Executed - The code has been executed and appears functional
- [ ] Environments configured - Any configuration updates have been added to the relevant environments
- [ ] Code documentation - Newly added code is sufficiently documented
- [ ] Readme - Necessary readme updated added

## Notes

Is there any other prudent information you would want to add?
```
